# Heritage Collection • Technical Blueprint

This repository contains the high-fidelity engineering archive of Chaniru Weerasinghe. It is designed as a zero-latency, production-grade library showcasing enterprise architectures, deep-learning engines, and high-performance digital systems.

## Core Philosophy

The "Heritage Collection" prioritizes absolute narrative control and performance integrity. By transitioning from dynamic API dependencies to a curated, hard-coded technical library, the system achieves near-instantaneous rendering and a professional authority that reflects the precision of the underlying projects.

## Technical Architecture

### Core Engine
- **Framework**: Next.js 15 (React 19 RC)
- **Primary Logic**: TypeScript
- **Styling**: Tailwind CSS 4.0 (Custom Precision Tokens)
- **Animation**: Framer Motion & GSAP Orchestration
- **Performance**: Zero-API dependency for project grids

### Key Systems
- **The Elite Collection**: A curated library of 25 enterprise-grade projects.
- **Technologies Built With**: A project-verified technical stack 1:1 match.
- **Security-First Design**: Glassmorphic UI with military-grade branding for proprietary systems.

## Development Workflow

### Requirements
- Node.js 18.17 or later
- npm / pnpm / yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/Chanii2024/Chaniru-Dev.git

# Install dependencies
npm install

# Initialize development server
npm run dev
```

## Security & Privacy

This repository showcases public snapshots of professional work. Proprietary architectures (such as the River Range Resort Staff ERP) are displayed with technical narratives only, maintaining the confidentiality of secure systems while demonstrating architectural mastery.
